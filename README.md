# Electric Air Purifying Face Mask

An open-source, Arduino-based electric face mask with built-in air purification system.

## Features
- Real-time air quality monitoring
- Adjustable fan speed
- Temperature and humidity sensing
- Battery level monitoring
- Quiet operation
- Multi-stage filtration

## Technical Specifications
- **Microcontroller**: Arduino Nano
- **Sensors**: DHT22, MQ-135
- **Fan**: Brushless DC (40mm)
- **Battery**: 18650 Li-ion (2000mAh)
- **Filtration**: HEPA + Activated Carbon
- **Runtime**: 4-6 hours

## Safety Features
- Over-current protection
- Battery monitoring
- Filter change reminders
- Emergency shutdown

## Applications
- Pandemic protection
- Pollution areas
- Allergen reduction
- Industrial safety
